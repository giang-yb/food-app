# Home Page Design Specification

## 1. Concept & Vision

**Bold, vibrant, and appetite-inducing** — The home page for a food & beverage delivery service that makes users hungry the moment they land. Combines high-energy promotional messaging with clean, modern layout. Fast, convenient, and visually mouth-watering.

**Tagline vibe:** "Tất cả đồ ăn bạn thèm, giao tận cửa trong 30 phút."

---

## 2. Design Language

### Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Primary | Coral Orange | `#FF6B35` | CTAs, highlights, primary buttons |
| Secondary | Fresh Green | `#4CAF50` | Fresh accent, category badges, success states |
| Accent | Golden Yellow | `#FFC107` | Deals, flash sale badges, star ratings |
| Background | Clean White | `#FFFFFF` | Main background |
| Surface | Light Gray | `#F8F9FA` | Card backgrounds, sections |
| Text Primary | Dark Charcoal | `#2D3436` | Headings, body text |
| Text Secondary | Medium Gray | `#636E72` | Captions, secondary info |

### Typography

- **Headings:** 'Poppins', sans-serif (Bold 600-700)
- **Body:** 'Inter', sans-serif (Regular 400, Medium 500)
- **Vietnamese support:** Font stack includes Vietnamese characters

### Visual Style

- **Border radius:** 12px for cards, 8px for buttons
- **Shadows:** Subtle elevation (`0 4px 20px rgba(0,0,0,0.08)`)
- **Images:** Rounded corners, subtle hover zoom effect
- **Spacing:** 8px base unit (8, 16, 24, 32, 48, 64)

### Motion Philosophy

- **Hover:** Scale up 1.02 + shadow lift (200ms ease-out)
- **Carousel:** Auto-play every 5s, smooth slide transition
- **Countdown:** Flip animation for timer digits
- **Page load:** Staggered fade-in for sections (100ms delay between)

## 4. Features & Interactions

### 4.1 Hero Banner Carousel

**Structure:**
- Full-width container (100vw), height: 500px desktop / 350px mobile
- Contains 3-4 slides with promotional content

**Promotional Messages (Random on each page load):**
1. "Miễn phí giao hàng cho đơn từ 100K"
2. "Flash Sale — Giảm 50% hôm nay!"
3. "Khách hàng mới — Giảm ngay 20%"
4. "Mua 2 tặng 1 — Áp dụng toàn bộ menu"

**Interactions:**
- Auto-play: 5 seconds per slide
- Manual navigation: Dots indicator + swipe on mobile
- CTA Button: "Khám phá ngay" → navigates to menu

**States:**
- Default: First slide visible
- Transition: 500ms slide animation
- Indicators: Active dot highlighted in primary color

### 4.2 Categories Showcase

**Categories:**
- 🍞 Bánh mì
- 🍚 Cơm
- 🥤 Nước ép
- 🥤 Sinh tố
- 🍦 Kem

**Card Design:**
- Icon/emoji centered (48px)
- Category name below
- Rounded rectangle, white background
- Subtle shadow on hover

**Interactions:**
- Hover: Scale 1.05, shadow lift
- Click: Navigate to category page (`/category/:slug`)

**Layout:**
- Horizontal scroll on mobile
- 5 columns on desktop
- Gap: 16px between cards

### 4.3 Featured Products Grid

**Content:**
- Display 8 products from various categories
- Mix of bestsellers and new items

**Product Card:**
```
┌─────────────────────┐
│   [Product Image]    │
│                     │
│  Product Name       │
│  Category Badge     │
│  ★★★★☆ (4.0)        │
│                     │
│  85.000 ₫           │
│  [Add to Cart]      │
└─────────────────────┘
```

**Interactions:**
- Hover: Image zoom 1.05, shadow lift
- Click: Navigate to product detail (`/product/:id`)
- "Add to Cart" button: Opens cart modal or adds directly

**Layout:**
- 4 columns desktop / 3 tablet / 2 mobile
- Load more button at bottom

### 4.4 Promotional Section (Flash Sale)

**Structure:**
- Background: Gradient orange (#FF6B35 → #FF8C42)
- White text, bold typography
- Countdown timer prominently displayed

**Content:**
- "FLASH SALE" header
- "Kết thúc sau:" + countdown (HH:MM:SS)
- Featured deal product
- "Xem tất cả" link

**Countdown Timer:**
- Flip animation on digit change
- Updates every second
- Shows "Đã kết thúc" when time reaches 0

## 5. Component Inventory

### HomePageComponent
- **Purpose:** Main container, orchestrates all sections
- **Children:** HeroBannerComponent, CategoriesShowcaseComponent, FeaturedProductsComponent, PromotionalSectionComponent

### HeroBannerComponent
| State | Visual |
|-------|--------|
| Loading | Skeleton placeholder (gray shimmer) |
| Loaded | Full carousel with active slide |
| Transitioning | Sliding animation between slides |
| Error | Fallback static banner |

### CategoryCardComponent
| State | Visual |
|-------|--------|
| Default | Icon + name, subtle shadow |
| Hover | Scale 1.05, deeper shadow |
| Active/Selected | Primary color border |
| Disabled | Grayed out (for unavailable categories) |

### ProductCardComponent
| State | Visual |
|-------|--------|
| Default | Image, name, price, rating |
| Hover | Image zoom, add-to-cart button appears |
| Loading | Skeleton |
| Out of Stock | "Hết hàng" badge overlay |
| In Cart | Checkmark indicator |

### CountdownTimerComponent
| State | Visual |
|-------|--------|
| Running | Flip animation digits |
| Final Hour | Red color, faster animation |
| Ended | "Đã kết thúc" message |

## 6. Technical Approach

### Framework & Architecture
- **Angular 17+** with Standalone Components
- **Signals** for state management
- **Angular Router** for navigation

### Project Structure

```
src/app/features/home/
├── pages/
│   └── home-page/
│       ├── home-page.component.ts
│       ├── home-page.component.html
│       └── home-page.component.scss
├── components/
│   ├── hero-banner/
│   ├── categories-showcase/
│   ├── featured-products/
│   └── promotional-section/
├── services/
│   └── home.service.ts
├── store/
│   └── home.store.ts
└── home.routes.ts
```

### Services

**HomeService**
- `getPromotionalBanners(): Observable<Banner[]>`
- `getCategories(): Observable<Category[]>`
- `getFeaturedProducts(): Observable<Product[]>`
- `getFlashSale(): Observable<FlashSale>`

### Store (Signal-based)

**HomeStore**
```typescript
interface HomeState {
  banners: Banner[];
  activeBannerIndex: number;
  categories: Category[];
  featuredProducts: Product[];
  flashSale: FlashSale | null;
  countdownSeconds: number;
  isLoading: boolean;
  error: string | null;
}
```

### Data Models

```typescript
interface Banner {
  id: string;
  imageUrl: string;
  title: string;
  subtitle: string;
  ctaText: string;
  ctaLink: string;
}

interface Category {
  id: string;
  name: string;
  icon: string; // emoji
  slug: string;
}

interface Product {
  id: string;
  name: string;
  price: number;
  originalPrice?: number;
  imageUrl: string;
  category: string;
  rating: number;
  reviewCount: number;
}

interface FlashSale {
  id: string;
  endTime: Date;
  products: Product[];
}
```

### Dependencies
- No external UI libraries (using custom components)
- Standalone Angular components
- CSS custom properties for theming
